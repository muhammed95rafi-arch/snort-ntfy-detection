# Snort IDS with Real-Time Alerting (ntfy Integration)

## 📌 Overview
This project demonstrates a practical Intrusion Detection System (IDS) setup using Snort to detect network attacks and send real-time alerts using ntfy.

The system is designed to simulate a basic Security Operations Center (SOC) environment where suspicious network activity is monitored and alerts are generated instantly.

---

## 🚀 Key Features
- Detection of Nmap TCP SYN scan attacks
- Real-time alert notifications using ntfy
- Custom Snort rule creation
- Lightweight and fast detection system
- SOC-style monitoring simulation

---

## 🛠️ Tools & Technologies
- Snort (Intrusion Detection System)
- Nmap (Attack simulation)
- ntfy (Real-time notification service)
- Linux (Ubuntu/Kali)
